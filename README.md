# 🕒 Simple Attendance Tracker

A clean and simple web application built with React to track entry and exit timings. All data is automatically saved directly to the browser's Local Storage, meaning records stay safe even if the page is refreshed or closed.

## ✨ Features
* **Single-Page Application:** Runs entirely in a single `index.html` file.
* **Real-Time Tracking:** Records precise entry and exit times.
* **Persistent Data:** Uses Local Storage so data is never lost on refresh.
* **Easy Reset:** Includes a one-click button to clear all records.
* **No Setup Required:** No backend or database configuration needed.

## 🚀 How to Run
1. Download or clone this repository.
2. Open the `index.html` file directly in any modern web browser (Chrome, Edge, Safari).
3. Start tracking!

## 🛠️ Technologies Used
* HTML5
* React 18 (via CDN)
* Babel (for in-browser JSX compilation)
* Local Storage API
